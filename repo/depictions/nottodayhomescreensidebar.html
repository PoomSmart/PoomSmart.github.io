Compatibility

  • Compatible with iOS 14.0 to 14.8.1

Description

  • Prevent Today View sidebar from being pinned on iPad homescreen. Widgets in Today View will still be accessible by swiping to the leftmost of the homescreen, just like how it is on iPad portrait or iPhone.

Source Code

  • %hook SBRootFolderView

- (bool)_shouldIgnoreOverscrollOnFirstPageForOrientation:(NSInteger)orientation {
	return true;
}

- (NSUInteger)sidebarAllowedOrientations {
	return 0;
}

%end
