Compatibility

  • Compatible with iOS 11.0 +

Description

  • Removes YouTube Music Premium upsell elements (banner, alerts, tab item).

Changelog

  • 1.0.2: Added more hooks for recent YouTube Music versions (contributed by @Tonwalter888)

- (id)sidePanelPromo { return nil; }
%end

// Hide Upgrade tab bar item at the bottom

- (YTPivotBarItemView *)itemView4 {

    view.navigationButton.hidden = YES;
    return nil;
}
%end
